Multiplicación de Matrices en PHP

La multiplicación de matrices es una operación fundamental en programación. Para calcular la matriz resultado, se multiplica cada fila de la primera matriz por cada columna de la segunda.

Ejemplo de matrices

Supongamos que tenemos una matriz A de tamaño 2x3 y una matriz B de tamaño 3x3. El resultado será una matriz C de tamaño 2x3.

Código en PHP


<?php
if (isset($_POST['multiplicar'])) {
    $a = $_POST['a']; // Matriz A (2x3)
    $b = $_POST['b']; // Matriz B (3x3)
    $c = array();

    for ($i = 0; $i < 2; $i++) {        // filas de A
        for ($j = 0; $j < 3; $j++) {    // columnas de B
            $c[$i][$j] = 0;
            for ($z = 0; $z < 3; $z++) { // recorrido común
                $c[$i][$j] += $a[$i][$z] * $b[$z][$j];
            }
        }
    }

    // Imprimir la matriz resultado
    echo "<h2>Matriz C (Resultado 2x3)</h2>";
    echo "<table border='1' cellpadding='10'>";
    for ($i = 0; $i < 2; $i++) {
        echo "<tr>";
        for ($j = 0; $j < 3; $j++) {
            echo "<td>" . $c[$i][$j] . "</td>";
        }
        echo "</tr>";
    }
    echo "</table>";
}
?>

Explicación del algoritmo

  • Se utilizan **tres ciclos anidados**: uno para recorrer filas, otro para columnas y otro para hacer las multiplicaciones parciales.
  • Cada posición C[i][j] se calcula multiplicando los elementos de la fila i de A por la columna j de B.
  • El resultado final es otra matriz cuyos valores dependen de todas las combinaciones de filas y columnas.

Ejemplo de salida

Si se ingresan los valores adecuados, la salida será algo como:


Matriz C (Resultado 2x3)
[ [58,  64,  69],
  [139, 154, 169] ]
