Ordenar un Arreglo Bidimensional en PHP

Un arreglo bidimensional es como una tabla con filas y columnas. Para ordenarlo, podemos aplicar algoritmos de ordenamiento sobre una de sus columnas.

Ejemplo de un arreglo bidimensional


<?php
$alumnos = [
    ["nombre" => "Ana", "edad" => 20],
    ["nombre" => "Luis", "edad" => 18],
    ["nombre" => "María", "edad" => 22],
    ["nombre" => "Pedro", "edad" => 19]
];
?>

Ordenando por una columna (ejemplo: edad)

Podemos usar el mismo algoritmo de selección que vimos con arreglos simples, pero comparando un campo en particular.


<?php
$n = count($alumnos);

for ($i = 0; $i < $n - 1; $i++) {
    $min = $i;

    for ($j = $i + 1; $j < $n; $j++) {
        if ($alumnos[$j]["edad"] < $alumnos[$min]["edad"]) {
            $min = $j;
        }
    }

    // Intercambiar filas completas
    $temp = $alumnos[$i];
    $alumnos[$i] = $alumnos[$min];
    $alumnos[$min] = $temp;
}

print_r($alumnos);
?>

Salida esperada

El arreglo se ordena por edad ascendente:


Array
(
    [0] => Array ( [nombre] => Luis [edad] => 18 )
    [1] => Array ( [nombre] => Pedro [edad] => 19 )
    [2] => Array ( [nombre] => Ana  [edad] => 20 )
    [3] => Array ( [nombre] => María [edad] => 22 )
)

Notas

  • Este método se puede aplicar para cualquier columna del arreglo.
  • En lugar de ciclos, también podrías usar la función usort() de PHP.
  • La idea es la misma: recorrer el arreglo y comparar un campo específico.
