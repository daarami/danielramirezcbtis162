Sentencia $_POST en PHP

La sentencia $_POST es una variable superglobal que permite recibir datos enviados desde un formulario HTML mediante el método POST.

¿Cómo funciona?

Cuando un formulario envía datos usando method="post", PHP los captura en $_POST como un array asociativo. La clave es el atributo name del campo, y el valor es lo que el usuario escribió.

Sintaxis básica

<?php
$valor = $_POST['nombre_del_campo'];
?>

Ejemplo práctico: Formulario

Formulario HTML que envía nombre y edad:

<form action="procesar.php" method="post">
  Nombre: <input type="text" name="nombre"><br>
  Edad: <input type="number" name="edad"><br>
  <input type="submit" value="Enviar">
</form>

Procesando los datos

<?php
// procesar.php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nombre = $_POST['nombre'];
    $edad = $_POST['edad'];
    echo "Hola, $nombre. Tienes $edad años.";
}
?>

Explicación paso a paso:

  • Se verifica que el formulario se envió mediante POST.
  • Se asigna cada valor del formulario a variables PHP usando $_POST['campo'].
  • Se puede procesar o mostrar la información según sea necesario.

Salida esperada (si el usuario escribe “Daniel” y “17”):

Hola, Daniel. Tienes 17 años.

Consideraciones importantes

  • Los datos enviados por POST no aparecen en la URL.
  • Siempre validar y sanear la información antes de usarla.
  • Ideal para datos sensibles como contraseñas o información privada.
