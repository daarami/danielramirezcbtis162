Sentencia do...while en PHP

El ciclo do...while ejecuta un bloque de código al menos una vez, y luego repite la ejecución mientras la condición siga siendo verdadera.

¿Cómo funciona?

A diferencia del while, el ciclo do...while evalúa la condición al final de cada iteración. Esto significa que el bloque se ejecuta al menos una vez, aunque la condición sea falsa desde el inicio.

Sintaxis básica

do {
  // código a ejecutar
} while (condición);

Ejemplo práctico

El siguiente ejemplo imprime los números del 1 al 5 utilizando do...while:

<?php
$numero = 1;

do {
  echo "Número: $numero <br>";
  $numero++;
} while ($numero <= 5);
?>

Explicación paso a paso:

  • Se inicia $numero con el valor 1.
  • El bloque dentro del do se ejecuta inmediatamente.
  • Luego se evalúa la condición $numero <= 5.
  • Si es verdadera, se repite el ciclo. Si no, termina.

Salida esperada:

Número: 1
Número: 2
Número: 3
Número: 4
Número: 5

Ejemplo 2: condición falsa desde el inicio

Aquí se puede ver cómo el do...while ejecuta el bloque una vez, incluso si la condición no se cumple. 

<?php
$numero = 10;

do {
  echo "Número: $numero <br>";
  $numero++;
} while ($numero <= 5);
?>

En este caso, aunque la condición es falsa (10 <= 5), el mensaje se imprime una vez antes de que termine el ciclo.

Comparación rápida

Ciclo Evalúa la condición Se ejecuta al menos una vez
while Antes del bloque No siempre
do...while Después del bloque Sí, siempre
