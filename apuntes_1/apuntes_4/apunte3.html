Sentencia for en PHP

La sentencia for es un tipo de ciclo que se utiliza cuando se conoce el número exacto de veces que se desea repetir un bloque de instrucciones.

¿Cómo funciona?

El ciclo for permite controlar tres elementos principales: la inicialización, la condición y la actualización. Se ejecuta mientras la condición sea verdadera.

Sintaxis básica

for (inicialización; condición; incremento) {
  // código a ejecutar
}

Partes del ciclo:

  • Inicialización: Se ejecuta una vez al inicio (por ejemplo, $i = 1).
  • Condición: Evalúa si el ciclo debe continuar (por ejemplo, $i <= 5).
  • Incremento: Cambia el valor de la variable en cada repetición (por ejemplo, $i++).

Ejemplo práctico

El siguiente ejemplo imprime los números del 1 al 5 utilizando for:

<?php
for ($i = 1; $i <= 5; $i++) {
  echo "Número: $i <br>";
}
?>

Explicación paso a paso:

  • Se inicia $i en 1.
  • El ciclo se repite mientras $i <= 5.
  • En cada vuelta, se imprime el número y luego $i aumenta en 1.

Salida esperada:

Número: 1
Número: 2
Número: 3
Número: 4
Número: 5

Ejemplo 2: Conteo regresivo

En este caso, el ciclo cuenta desde 5 hasta 1:

<?php
for ($i = 5; $i >= 1; $i--) {
  echo "Cuenta atrás: $i <br>";
}
?>

Salida esperada:

Cuenta atrás: 5
Cuenta atrás: 4
Cuenta atrás: 3
Cuenta atrás: 2
Cuenta atrás: 1

Comparación con otros ciclos

Ciclo Evaluación de condición Ideal para
while Antes del bloque Repeticiones indefinidas
do...while Después del bloque Ejecutar al menos una vez
for Antes del bloque Repeticiones conocidas
