Sentencia while en PHP

La sentencia while permite ejecutar un bloque de código repetidamente mientras una condición sea verdadera.

¿Cómo funciona?

Primero se evalúa una condición. Si el resultado es true, se ejecutan las instrucciones dentro del ciclo. Cuando la condición deja de cumplirse, el bucle termina.

Importante: Si la condición nunca cambia a falsa, el ciclo será infinito.

Sintaxis básica

while (condición) {
  // código a ejecutar mientras la condición sea verdadera
}

Ejemplo práctico

El siguiente ejemplo muestra cómo imprimir los números del 1 al 5 utilizando while:

<?php
$numero = 1;

while ($numero <= 5) {
  echo "Número: $numero <br>";
  $numero++; // aumenta el valor para evitar un bucle infinito
}
?>

Explicación paso a paso:

  • Se inicia la variable $numero con el valor 1.
  • El while verifica si $numero <= 5.
  • Si la condición es verdadera, se ejecuta el bloque y se imprime el número.
  • Luego se incrementa $numero con $numero++.
  • Cuando $numero llega a 6, la condición ya no se cumple y el ciclo termina.

Salida esperada:

Número: 1
Número: 2
Número: 3
Número: 4
Número: 5

Otro ejemplo: contador regresivo

<?php
$contador = 5;

while ($contador >= 1) {
  echo "Cuenta atrás: $contador <br>";
  $contador--;
}
?>

Este código imprime una cuenta regresiva desde 5 hasta 1, restando 1 en cada repetición.
