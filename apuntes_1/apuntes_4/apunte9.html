Sentencia PDO en PHP

PDO (PHP Data Objects) es una interfaz que permite conectarse y trabajar con bases de datos de manera segura y uniforme, independientemente del motor de base de datos que se use.

¿Cómo funciona?

PDO permite conectarse a una base de datos, ejecutar consultas y manejar resultados. Además, ofrece soporte para sentencias preparadas, lo que previene ataques de inyección SQL.

Sintaxis básica

$pdo = new PDO("mysql:host=localhost;dbname=nombreBD", "usuario", "contraseña");

Ejemplo práctico: Conexión a MySQL

<?php
try {
  $pdo = new PDO("mysql:host=localhost;dbname=miBD", "root", "");
  $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
  echo "Conexión exitosa";
} catch(PDOException $e) {
  echo "Error de conexión: " . $e->getMessage();
}
?>

Salida esperada (si la conexión es correcta):

Conexión exitosa

Ejemplo práctico: Consulta con PDO

Obtener datos de una tabla llamada usuarios:

<?php
$consulta = $pdo->query("SELECT nombre, email FROM usuarios");

while($fila = $consulta->fetch(PDO::FETCH_ASSOC)) {
  echo "Nombre: " . $fila['nombre'] . " - Email: " . $fila['email'] . "<br>";
}
?>

Explicación paso a paso:

  • $pdo->query() ejecuta la consulta SQL.
  • fetch(PDO::FETCH_ASSOC) obtiene cada fila como un arreglo asociativo.
  • El ciclo while recorre todas las filas y las muestra.

Ejemplo práctico: Sentencias preparadas

Insertar datos de manera segura:

<?php
$sentencia = $pdo->prepare("INSERT INTO usuarios (nombre, email) VALUES (:nombre, :email)");
$sentencia->execute([
  ':nombre' => 'Daniel',
  ':email' => 'daniel@mail.com'
]);
echo "Usuario insertado correctamente";
?>

Consideraciones importantes

  • Siempre usar sentencias preparadas para prevenir inyección SQL.
  • Pueden usarse con diferentes motores de base de datos (MySQL, SQLite, PostgreSQL, etc.).
  • El manejo de errores se realiza con excepciones usando try/catch.
