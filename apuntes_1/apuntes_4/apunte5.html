Arrays en PHP

Un array es una estructura de datos que permite almacenar múltiples valores bajo un mismo nombre y acceder a ellos mediante índices o claves.

Tipos de arrays

  • Array indexado: Los elementos tienen un índice numérico automático o definido por el usuario.
  • Array asociativo: Los elementos se acceden mediante claves definidas por el usuario.
  • Array multidimensional: Contiene uno o más arrays dentro de otro array.

Sintaxis básica

// Array indexado
$frutas = array("Manzana", "Banana", "Cereza");

// Array asociativo
$edades = array("Daniel" => 17, "Ana" => 18);

// Array multidimensional
$personas = array(
    array("nombre" => "Daniel", "edad" => 17),
    array("nombre" => "Ana", "edad" => 18)
);

Ejemplo práctico: Array indexado

<?php
$colores = array("Rojo", "Verde", "Azul");

echo $colores[0]; // Rojo
echo $colores[2]; // Azul
?>

Salida esperada:

Rojo
Azul

Ejemplo práctico: Array asociativo

<?php
$edades = array("Daniel" => 17, "Ana" => 18);

echo "Daniel tiene " . $edades['Daniel'] . " años.";
echo "<br>";
echo "Ana tiene " . $edades['Ana'] . " años.";
?>

Salida esperada:

Daniel tiene 17 años.
Ana tiene 18 años.

Ejemplo práctico: Array multidimensional

<?php
$personas = array(
    array("nombre" => "Daniel", "edad" => 17),
    array("nombre" => "Ana", "edad" => 18)
);

echo $personas[0]['nombre'] . " tiene " . $personas[0]['edad'] . " años.<br>";
echo $personas[1]['nombre'] . " tiene " . $personas[1]['edad'] . " años.";
?>

Salida esperada:

Daniel tiene 17 años.
Ana tiene 18 años.

Funciones útiles con arrays

  • count($array) – Devuelve el número de elementos.
  • array_push($array, $valor) – Agrega un elemento al final.
  • array_pop($array) – Elimina el último elemento.
  • array_keys($array) – Devuelve todas las claves de un array asociativo.
  • array_values($array) – Devuelve todos los valores de un array asociativo.
