Sentencia $_GET en PHP

$_GET es una variable superglobal que se utiliza para capturar datos enviados mediante la URL, generalmente desde formularios con el método get.

¿Cómo funciona?

Los datos enviados se agregan a la URL en forma de parámetros (query string) y se acceden a través de la variable $_GET mediante la clave asignada a cada campo.

Sintaxis básica

<?php
$valor = $_GET['clave'];
?>

Ejemplo práctico: Formulario simple

Formulario HTML que envía datos por GET:

<form action="procesar.php" method="get">
  Nombre: <input type="text" name="nombre"><br>
  Edad: <input type="number" name="edad"><br>
  <input type="submit" value="Enviar">
</form>

Archivo procesar.php que recibe los datos:

<?php
$nombre = $_GET['nombre'];
$edad = $_GET['edad'];

echo "Hola $nombre, tienes $edad años.";
?>

Salida esperada (si se envía nombre=Daniel y edad=17):

Hola Daniel, tienes 17 años.

Consideraciones importantes

  • Los datos enviados por GET son visibles en la URL.
  • No es seguro enviar información sensible mediante GET.
  • Es útil para pasar parámetros entre páginas y para enlaces que requieren información dinámica.
  • Se puede combinar con isset() para verificar si un parámetro existe:
<?php
if(isset($_GET['nombre'])) {
  echo "Nombre recibido: " . $_GET['nombre'];
} else {
  echo "No se recibió nombre.";
}
?>
