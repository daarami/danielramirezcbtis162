Sentencia isset() en PHP

isset() es una función que permite verificar si una variable está definida y no es null. Es útil para validar datos antes de usarlos.

¿Cómo funciona?

La función devuelve true si la variable existe y tiene un valor diferente de null, y false en caso contrario.

Sintaxis básica

isset($variable);

Ejemplo práctico: Verificar una variable

<?php
$nombre = "Daniel";

if(isset($nombre)) {
  echo "La variable \$nombre está definida.";
} else {
  echo "La variable \$nombre no está definida.";
}
?>

Salida esperada:

La variable $nombre está definida.

Ejemplo práctico con formularios

Validando datos enviados por GET:

<?php
if(isset($_GET['edad'])) {
  echo "Edad recibida: " . $_GET['edad'];
} else {
  echo "No se recibió edad.";
}
?>

Salida esperada (si se envía edad=17):

Edad recibida: 17

Consideraciones importantes

  • No genera error si la variable no existe, a diferencia de acceder directamente a la variable.
  • Se puede usar con múltiples variables: isset($a, $b, $c) devuelve true solo si todas existen y no son null.
  • Es común usar isset() junto con $_GET o $_POST para validar entradas antes de procesarlas.
