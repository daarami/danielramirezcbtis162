Cómo visualizar archivos

Visualizar tus archivos te permite comprobar que tu código funciona correctamente y ver los resultados de tu proyecto web.

Formas de visualizar archivos

  • Desde el explorador de archivos: Haz doble clic sobre el archivo HTML, PHP o CSS para abrirlo en tu navegador predeterminado.
  • Desde Visual Studio Code: Usa extensiones como Live Server para abrir un servidor local y ver los cambios en tiempo real.
  • En el navegador: Puedes arrastrar y soltar tu archivo HTML o abrirlo directamente desde Archivo → Abrir archivo.

Ver archivos PHP

Los archivos PHP no se pueden abrir directamente en el navegador como HTML, ya que necesitan un servidor para procesar el código:

  • Instala un servidor local como MAMP, XAMPP o WAMP.
  • Coloca tus archivos PHP dentro de la carpeta del servidor (por ejemplo, htdocs en XAMPP o Sites en MAMP).
  • Abre el navegador y escribe http://localhost/nombre-de-tu-archivo.php para ver el resultado.

Visualización en tiempo real

  • Con VS Code y Live Server, cada vez que guardes cambios, el navegador se actualizará automáticamente.
  • Esto es útil para HTML, CSS y JS, mientras que para PHP necesitarás el servidor local activo.
