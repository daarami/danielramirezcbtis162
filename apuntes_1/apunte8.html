Cómo guardar archivos en Visual Studio Code

Guardar archivos correctamente es esencial para no perder tus avances y mantener tu proyecto organizado.

Métodos para guardar archivos

  • Guardar individual: Presiona Ctrl + S (Windows/Linux) o Cmd + S (macOS) para guardar el archivo activo.
  • Guardar todos los archivos abiertos: Presiona Ctrl + K S (Windows/Linux) o Cmd + Option + S (macOS).
  • Usando el menú: Ve a Archivo → Guardar o Archivo → Guardar todo.

Guardar con nombre o en otra ubicación

  • Para cambiar el nombre del archivo o guardarlo en otra carpeta, usa Archivo → Guardar como....
  • Es recomendable organizar tus archivos en carpetas dentro de tu proyecto, por ejemplo: Trabajos Modulo.

Extensiones y formato de archivo

  • Asegúrate de guardar tus archivos con la extensión correcta: .php para PHP, .html para HTML, .css para CSS.
  • VS Code detecta automáticamente la extensión y aplica el resaltado de sintaxis correspondiente.
