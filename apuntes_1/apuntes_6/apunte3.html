Bases de datos (AppSheet)

Formas principales de crear y conectar las bases de datos que usa AppSheet para almacenar la información.

Excel (.xlsx)

Crea tus tablas en un archivo Excel (.xlsx) y súbelo a Google Drive. Cada hoja del archivo puede funcionar como una tabla que AppSheet leerá.

Google Sheets

Crea las tablas directamente en Google Sheets dentro de tu Google Drive. Cada hoja representa una tabla y se edita en línea.

Base de datos nativa

También puedes diseñar las tablas directamente dentro del editor de AppSheet, sin depender de archivos externos. Este sistema nativo aún está en fase BETA y tiene algunas limitaciones:

  • Máximo de 10,000 filas por tabla.
  • Hasta 20 tablas por base de datos.

Conexión con Google Drive

Si usas Excel o Google Sheets, es necesario vincular AppSheet con tu cuenta de Google Drive para que la app pueda acceder y usar las tablas como base de datos.
