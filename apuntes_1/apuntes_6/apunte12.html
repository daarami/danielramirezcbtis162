TIPOS DE DATOS — Enumerados (3)

En esta categoría se incluyen tipos de datos que conectan tablas o permiten valores booleanos dentro de la app.

Ref

Se usa para crear relaciones entre tablas o registros. Este tipo permite enlazar un valor con otro registro existente, facilitando la navegación y la gestión de datos relacionados.
Consulta también: Referencias entre tablas.

Yes/No

Tipo de dato booleano que permite únicamente dos valores: Yes (verdadero) o No (falso). Es útil para activar opciones, marcar estados o validar condiciones simples.
