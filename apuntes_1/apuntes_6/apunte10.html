TIPOS DE DATOS — Enumerados (1)

Tipos de datos que permiten seleccionar valores de listas o conjuntos predefinidos. Muy útiles para mantener la consistencia de los datos dentro de una app.

Color

Permite añadir entradas de color en la aplicación a través de un conjunto limitado de valores estándar: Black, Blue, Green, Orange, Purple, Red, Yellow y White. Ideal para etiquetas o indicadores visuales dentro de los registros.

Enum

Permite al usuario seleccionar un único valor de una lista de opciones predefinidas. Este tipo de campo genera un menú desplegable para facilitar la elección.
