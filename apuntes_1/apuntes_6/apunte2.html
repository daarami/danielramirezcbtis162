Elementos importantes de AppSheet

Para crear una aplicación funcional en AppSheet, es necesario comprender los elementos básicos que la sustentan.

1. Base de datos

AppSheet puede conectarse a diferentes tipos de fuentes de datos, como:

  • MySQL
  • Excel
  • Google Sheets
  • Bases de datos nativas de la web

2. Tablas y relaciones

La información se organiza en tablas, cada una con columnas (campos) que representan un tipo de dato específico. Las tablas pueden estar relacionadas entre sí por medio de una columna común, por ejemplo un ID o una clave foránea.

3. Tipos de datos

Cada columna debe tener un tipo de dato adecuado al contenido que almacenará. Algunos ejemplos comunes son:

  • Texto: para nombres o descripciones.
  • Número: para cantidades o precios.
  • Fecha/Hora: para registrar momentos específicos.
  • Imagen: para guardar fotos o archivos visuales.

Estos elementos son la base de cualquier app creada en AppSheet, ya que determinan cómo se almacena, relaciona y muestra la información.
