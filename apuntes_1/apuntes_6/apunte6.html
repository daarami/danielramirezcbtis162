TIPOS DE DATOS — Números

AppSheet permite varios tipos de datos numéricos según el tipo de valor que se quiera guardar en una columna.

Decimal

Representa un número real con parte fraccionaria, también conocido como número de punto flotante.

Number

Guarda números enteros. Internamente se almacena como un entero de 54 bits, con un rango de -2^53 a 2^53.

Percent

Es un valor decimal que representa un porcentaje. 0% se almacena como 0.00 y 100% como 1.00.

Price

Es un valor decimal que representa una cantidad monetaria.
