TIPOS DE DATOS — Texto

AppSheet ofrece varios tipos de texto para columnas según el uso y la longitud del contenido.

LongText

Permite una o más líneas de texto. Útil para descripciones, comentarios o notas largas.

Name

Diseñado para nombres (personas, lugares, entidades). Optimiza la validación y el formato de nombres.

Text

Texto de una sola línea. Ideal para títulos, etiquetas o valores cortos.
