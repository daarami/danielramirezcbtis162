TIPOS DE DATOS — Otros

AppSheet también ofrece tipos de datos especiales que amplían las funciones básicas de la app, permitiendo conectar enlaces, cambiar entre apps o combinar varias columnas en una sola clave.

1. URL

Representa una dirección web. Un valor de tipo URL puede tener dos partes:

  • Una dirección completa, como https://es.wikipedia.org.
  • Un texto de hipervínculo opcional, por ejemplo: “Haz clic aquí para visitar Wikipedia”.

Puedes definir ambos valores desde la pestaña Datos > Columnas en el editor de AppSheet. Para el texto de visualización, cambia la expresión del atributo Visualización.

2. App

Este tipo de dato se usa para crear enlaces internos entre aplicaciones AppSheet, o para navegar hacia una vista diferente dentro de la misma aplicación. Es útil cuando tu proyecto tiene múltiples apps conectadas o flujos de navegación personalizados.

3. MultiColumnKey

Es un campo compuesto que une varios valores de otras columnas con el propósito de formar una sola clave. Se utiliza cuando un registro necesita identificarse mediante una combinación de datos (por ejemplo, ID del cliente + Fecha del pedido).
