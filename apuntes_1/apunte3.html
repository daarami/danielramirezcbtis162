Cómo instalar MAMP para servidor PHP

MAMP es una herramienta que permite instalar fácilmente un servidor local con Apache, MySQL y PHP en Windows o macOS, ideal para desarrollo web.


Pasos de instalación

  1. Descargar MAMP: Ve a https://www.mamp.info/ y descarga la versión correspondiente a tu sistema operativo.
  2. Ejecutar el instalador: Abre el archivo descargado y sigue las instrucciones en pantalla. Puedes dejar las configuraciones predeterminadas.
  3. Abrir MAMP: Una vez instalado, abre la aplicación. Verás los botones para iniciar Apache y MySQL.
  4. Configurar puertos (opcional): Por defecto, MAMP usa el puerto 8888 para Apache y 8889 para MySQL. Puedes cambiarlos en las preferencias si es necesario.
  5. Probar el servidor: Haz clic en "Start Servers" y luego abre tu navegador en http://localhost:8888. Deberías ver la página de inicio de MAMP.
  6. Crear proyectos PHP: Coloca tus archivos PHP en la carpeta htdocs dentro del directorio de MAMP. Luego accede a ellos desde el navegador, por ejemplo: http://localhost:8888/mi-proyecto/index.php.

Ventajas de usar MAMP

  • Instalación rápida y sin complicaciones.
  • Compatible con Windows y macOS.
  • Incluye Apache, MySQL y PHP configurados automáticamente.
  • Ideal para pruebas locales y desarrollo web antes de subir a un servidor real.
