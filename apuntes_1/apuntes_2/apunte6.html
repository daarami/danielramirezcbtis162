Precedencia de Operadores en PHP

La precedencia de operadores indica el orden en que se evalúan los operadores dentro de una expresión. Esto es importante porque el mismo resultado puede cambiar dependiendo de cómo se evalúe.

¿Cómo funciona?

Los operadores con mayor precedencia se ejecutan primero. Cuando varios operadores tienen la misma precedencia, se evalúan de acuerdo a su asociatividad (izquierda a derecha o derecha a izquierda).

Ejemplo básico


<?php
  echo 2 + 3 * 4; 
  // Resultado: 14
  // Primero se calcula 3 * 4 = 12
  // Luego se suma 2 + 12 = 14

  echo (2 + 3) * 4; 
  // Resultado: 20
  // Los paréntesis cambian la prioridad
?>

Precedencia común en PHP

Operador Descripción Asociatividad
() Paréntesis -
++ -- Incremento / Decremento Derecha a izquierda
* / % Multiplicación, división, módulo Izquierda a derecha
+ - Suma, resta Izquierda a derecha
. Concatenación de cadenas Izquierda a derecha
< > <= >= == === != !== Comparaciones Izquierda a derecha
&& AND lógico Izquierda a derecha
|| OR lógico Izquierda a derecha
= += -= *= /= .= Asignación Derecha a izquierda
