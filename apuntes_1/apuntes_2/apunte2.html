Concatenación en PHP

La concatenación es el proceso de unir dos o más cadenas de texto. En PHP se utiliza el operador . (punto) para realizar esta operación.

Sintaxis

El operador de concatenación en PHP se escribe con un punto:


<?php
  $texto1 = "Hola";
  $texto2 = "Mundo";
  $resultado = $texto1 . " " . $texto2;
  echo $resultado; // Muestra: Hola Mundo
?>

Concatenación combinada

También se puede usar .= para añadir contenido a una variable existente:


<?php
  $mensaje = "Bienvenidos";
  $mensaje .= " a PHP";
  echo $mensaje; // Muestra: Bienvenidos a PHP
?>

Usos comunes

  • Construir mensajes personalizados.
  • Unir variables con texto en una sola salida.
  • Generar cadenas dinámicas como rutas, consultas SQL o HTML.
Nota: No confundas el punto (.) de concatenación con el signo +, que solo sirve para sumar números.
