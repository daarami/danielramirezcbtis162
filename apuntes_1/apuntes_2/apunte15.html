Estructuras de Control en PHP

Las estructuras de control permiten modificar el flujo de ejecución de un programa según condiciones o repeticiones.

Condicionales

Se utilizan para ejecutar código solo si se cumple una condición:


<?php
  $edad = 18;

  if ($edad >= 18) {
      echo "Eres mayor de edad";
  } else {
      echo "Eres menor de edad";
  }

  // switch
  $dia = "Lunes";
  switch ($dia) {
      case "Lunes":
          echo "Comienza la semana";
          break;
      case "Viernes":
          echo "Último día laboral";
          break;
      default:
          echo "Día normal";
  }
?>

Bucles

Permiten repetir código mientras se cumpla una condición o un número determinado de veces:


<?php
  // while
  $i = 1;
  while ($i <= 5) {
      echo $i;
      $i++;
  }

  // do...while
  $j = 1;
  do {
      echo $j;
      $j++;
  } while ($j <= 5);

  // for
  for ($k = 1; $k <= 5; $k++) {
      echo $k;
  }

  // foreach (para arrays)
  $frutas = ["Manzana", "Banana", "Cereza"];
  foreach ($frutas as $fruta) {
      echo $fruta;
  }
?>
