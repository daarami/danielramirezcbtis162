Operadores de Tipo en PHP

Los operadores de tipo permiten comprobar o convertir el tipo de un valor en PHP, asegurando un manejo correcto de datos.

Operadores principales

  • instanceof → Verifica si un objeto es una instancia de una clase determinada.
  • (int), (float), (string), (bool), (array), (object), (unset) → Cast o conversión de tipo explícita.

Ejemplos de uso


<?php
  // Conversión de tipos
  $num = "123";
  $numInt = (int)$num; // convierte a entero
  var_dump($numInt);   // int(123)

  $val = 3.14;
  $valStr = (string)$val; 
  var_dump($valStr);   // string(4) "3.14"

  // Verificación de instancia
  class Persona {}
  $p = new Persona();
  if ($p instanceof Persona) {
      echo "Es una instancia de Persona";
  }
?>
