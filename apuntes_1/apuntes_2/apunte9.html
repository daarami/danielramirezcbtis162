Operadores de Comparación en PHP

Los operadores de comparación se utilizan para comparar valores y devolver un resultado booleano (true o false). Son esenciales en estructuras condicionales y bucles.

Tipos de operadores de comparación

PHP ofrece varios operadores de comparación que permiten evaluar igualdad, desigualdad, y relaciones entre valores.

Ejemplos básicos


<?php
  $a = 5;
  $b = 10;

  var_dump($a == $b);   // false → Compara si son iguales
  var_dump($a != $b);   // true  → Compara si son diferentes
  var_dump($a < $b);    // true  → Compara si $a es menor que $b
  var_dump($a >= $b);   // false → Compara si $a es mayor o igual que $b
?>

Operadores de comparación más comunes

Operador Descripción Ejemplo
== Igualdad de valores $x == $y
=== Identidad (igual valor y mismo tipo) $x === $y
!= Diferencia de valores $x != $y
!== No idéntico (diferente valor o tipo) $x !== $y
< Menor que $x < $y
> Mayor que $x > $y
<= Menor o igual que $x <= $y
>= Mayor o igual que $x >= $y
<=> Operador combinado "spaceship" (devuelve -1,0,1) $x <=> $y
