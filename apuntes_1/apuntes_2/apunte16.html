Sentencia IF en PHP

La sentencia if permite ejecutar un bloque de código únicamente si se cumple una condición.

Sintaxis básica


<?php
if (condición) {
    // Código a ejecutar si la condición es verdadera
} 
?>

Uso de ELSE

Si la condición no se cumple, podemos usar else para ejecutar otro bloque:


<?php
$edad = 17;

if ($edad >= 18) {
    echo "Eres mayor de edad";
} else {
    echo "Eres menor de edad";
}
?>

Uso de ELSEIF

Para múltiples condiciones:


<?php
$nota = 85;

if ($nota >= 90) {
    echo "Excelente";
} elseif ($nota >= 70) {
    echo "Bueno";
} else {
    echo "Necesita mejorar";
}
?>
