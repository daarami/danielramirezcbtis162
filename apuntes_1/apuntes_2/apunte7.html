Operadores Aritméticos en PHP

Los operadores aritméticos permiten realizar operaciones matemáticas básicas en PHP, como suma, resta, multiplicación y división. Son fundamentales para trabajar con números dentro de programas.

Lista de operadores

Operador Descripción Ejemplo Resultado
+ Suma 5 + 3 8
- Resta 10 - 4 6
* Multiplicación 6 * 2 12
/ División 15 / 3 5
% Módulo (resto de la división) 10 % 3 1
** Exponenciación 2 ** 4 16

Ejemplo en PHP


<?php
  $a = 10;
  $b = 3;

  echo $a + $b; // 13
  echo $a - $b; // 7
  echo $a * $b; // 30
  echo $a / $b; // 3.333...
  echo $a % $b; // 1
  echo $a ** $b; // 1000
?>

Tips importantes

  • La división entre enteros puede dar como resultado un número decimal.
  • El operador módulo % es muy útil para saber si un número es par o impar.
  • PHP también soporta la exponenciación (**), que no existe en muchos otros lenguajes.
