Expresiones en PHP

Una expresión en PHP es cualquier combinación de valores, variables y operadores que produce un resultado. Son la base de las instrucciones en un programa.

Tipos de expresiones

  • Numéricas: combinan números y operadores aritméticos.
  • De cadena: trabajan con texto y operadores como la concatenación.
  • Lógicas: devuelven un valor verdadero (true) o falso (false).
  • De asignación: guardan valores en variables.

Ejemplos básicos


<?php
  // Expresión numérica
  $suma = 5 + 3;   // Resultado: 8

  // Expresión de cadena
  $nombre = "Daniel";
  $mensaje = "Hola " . $nombre;  // "Hola Daniel"

  // Expresión lógica
  $edad = 18;
  $esMayor = ($edad >= 18); // true

  // Expresión de asignación
  $x = 10;  // Asigna el valor 10 a la variable $x
?>

Expresiones en condiciones

Muchas veces las expresiones se usan en estructuras como if, donde el programa decide qué hacer en base a su resultado:


<?php
  $calificacion = 85;

  if ($calificacion >= 70) {
      echo "Aprobado";
  } else {
      echo "Reprobado";
  }
?>

¿Por qué son importantes?

  • Son la unidad básica de trabajo en cualquier lenguaje de programación.
  • Permiten calcular resultados a partir de datos.
  • Se combinan con variables, operadores y funciones para construir programas completos.
