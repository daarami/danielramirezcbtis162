Operadores en PHP

Los operadores son símbolos especiales que permiten realizar operaciones con variables y valores. En PHP existen varios tipos, cada uno con un propósito específico.

Principales tipos de operadores

  • Aritméticos: realizan operaciones matemáticas.
  • De asignación: guardan valores en variables.
  • De comparación: comparan valores y devuelven true o false.
  • Lógicos: combinan condiciones lógicas.
  • De concatenación: unen cadenas de texto.

Ejemplos básicos


<?php
  // Operadores aritméticos
  $a = 10;
  $b = 3;
  echo $a + $b; // 13
  echo $a % $b; // 1 (módulo)

  // Operadores de asignación
  $x = 5;
  $x += 3; // $x ahora vale 8

  // Operadores de comparación
  var_dump($a == $b);  // false
  var_dump($a > $b);   // true

  // Operadores lógicos
  $mayorEdad = true;
  $tieneINE = false;
  var_dump($mayorEdad && $tieneINE); // false

  // Operador de concatenación
  $nombre = "Daniel";
  $saludo = "Hola " . $nombre;
  echo $saludo; // "Hola Daniel"
?>

Tabla de operadores más comunes

Operador Tipo Ejemplo Resultado
+ Aritmético 5 + 3 8
== Comparación 5 == "5" true
=== Comparación estricta 5 === "5" false
&& Lógico true && false false
. Concatenación "Hola " . "Mundo" Hola Mundo

¿Por qué son importantes?

  • Permiten manipular y combinar datos.
  • Son la base de cálculos, comparaciones y condiciones.
  • Se usan en prácticamente todas las expresiones en PHP.
