Variables en PHP

Una variable es un espacio en memoria donde se guarda información que puede cambiar durante la ejecución de un programa. En PHP, todas las variables comienzan con el signo $.

Reglas básicas

  • Deben comenzar con $ seguido de un nombre.
  • El nombre puede contener letras, números y guiones bajos, pero no puede empezar con un número.
  • PHP distingue entre mayúsculas y minúsculas: $edad y $Edad son diferentes.

Ejemplo básico


<?php
  $nombre = "Daniel";
  $edad = 18;
  $activo = true;

  echo "Hola, mi nombre es $nombre y tengo $edad años.";
?>

Tipos de valores que puede guardar

  • String: texto, se escribe entre comillas ("Hola" o 'Mundo').
  • Integer: números enteros (10, -5).
  • Float: números decimales (3.14).
  • Boolean: valores lógicos (true o false).
  • Array: conjunto de valores almacenados en una sola variable.

Ejemplo con varios tipos


<?php
  $precio = 25.50;
  $disponible = false;
  $colores = array("rojo", "verde", "azul");

  echo "Precio: $precio"; // Muestra: Precio: 25.5
?>

¿Por qué son importantes?

  • Permiten reutilizar valores sin escribirlos varias veces.
  • Hacen que el código sea más flexible y fácil de modificar.
  • Ayudan a organizar la información dentro de un programa.
