Sintaxis Alternativa de Estructuras de Control en PHP

La sintaxis alternativa es útil cuando mezclamos PHP y HTML, ya que reemplaza las llaves { } por dos puntos : y palabras clave de cierre como endif;.

Ejemplo con IF


<?php
$edad = 20;

if ($edad >= 18):
    echo "Eres mayor de edad";
else:
    echo "Eres menor de edad";
endif;
?>

Ejemplo con ELSEIF


<?php
$nota = 75;

if ($nota >= 90):
    echo "Excelente";
elseif ($nota >= 70):
    echo "Bueno";
else:
    echo "Necesita mejorar";
endif;
?>

Ejemplo con FOR


<?php
for ($i = 1; $i <= 5; $i++):
    echo "<p>Número $i</p>";
endfor;
?>

Ejemplo con FOREACH


<?php
$frutas = ["Manzana", "Banana", "Cereza"];

foreach ($frutas as $fruta):
    echo "<p>$fruta</p>";
endforeach;
?>
