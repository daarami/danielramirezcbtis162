Operadores Lógicos en PHP

Los operadores lógicos se utilizan para combinar o invertir condiciones booleanas (true o false). Son muy útiles en estructuras de control como if o while.

Tipos de operadores lógicos

PHP incluye varios operadores lógicos:

  • && (AND lógico) → verdadero si ambas condiciones son verdaderas.
  • || (OR lógico) → verdadero si al menos una condición es verdadera.
  • ! (NOT lógico) → invierte el valor de la condición.
  • xor (XOR lógico) → verdadero si solo una de las condiciones es verdadera.
  • and y or → similar a && y || pero con menor precedencia.

Ejemplos


<?php
  $a = true;
  $b = false;

  // AND lógico
  var_dump($a && $b); // false

  // OR lógico
  var_dump($a || $b); // true

  // NOT lógico
  var_dump(!$b); // true

  // XOR lógico
  var_dump($a xor $b); // true

  // Diferencia entre && y and
  $x = false;
  $y = true;
  $result = $x && $y; // evalúa antes que asignación
  var_dump($result); // false
?>
