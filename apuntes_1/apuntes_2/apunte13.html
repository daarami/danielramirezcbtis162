Operadores de Arrays en PHP

PHP ofrece operadores especiales para comparar y combinar arrays de manera sencilla.

Operadores principales

  • + → Unión: combina dos arrays, los valores de claves duplicadas del primer array se mantienen.
  • == → Igualdad: devuelve true si los arrays tienen los mismos pares clave-valor, sin importar el orden.
  • === → Identidad: devuelve true si los arrays tienen los mismos pares clave-valor en el mismo orden y del mismo tipo.
  • != o <> → Diferencia: true si los arrays no son iguales.
  • !== → No idéntico: true si los arrays no son idénticos.

Ejemplos


<?php
  $a = array("a" => "rojo", "b" => "verde");
  $b = array("c" => "azul", "b" => "verde");

  // Unión
  $c = $a + $b;
  print_r($c);
  /* Resultado:
    Array
    (
      [a] => rojo
      [b] => verde
      [c] => azul
    )
  */

  // Comparación
  var_dump($a == $b);  // false
  var_dump($a === $b); // false
?>
