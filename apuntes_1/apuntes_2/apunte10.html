Operadores de Incremento y Decremento en PHP

Los operadores de incremento y decremento permiten aumentar o disminuir el valor de una variable en 1. Son muy útiles para bucles y contadores.

Tipos de operadores

Existen dos formas de utilizarlos: prefijo y postfijo, lo que afecta el momento en que se modifica el valor.

Ejemplos básicos


<?php
  $x = 5;

  // Incremento
  echo ++$x; // prefijo: primero aumenta, luego imprime → 6
  echo $x++; // postfijo: primero imprime, luego aumenta → 6
  echo $x;   // 7 (después de la operación postfija)

  // Decremento
  echo --$x; // prefijo: primero disminuye, luego imprime → 6
  echo $x--; // postfijo: primero imprime, luego disminuye → 6
  echo $x;   // 5
?>

Tabla de operadores

Operador Descripción Ejemplo
++$x Incremento prefijo (suma 1 antes de usar el valor) $x = 5; echo ++$x; // 6
$x++ Incremento postfijo (suma 1 después de usar el valor) $x = 5; echo $x++; // 5
--$x Decremento prefijo (resta 1 antes de usar el valor) $x = 5; echo --$x; // 4
$x-- Decremento postfijo (resta 1 después de usar el valor) $x = 5; echo $x--; // 5
