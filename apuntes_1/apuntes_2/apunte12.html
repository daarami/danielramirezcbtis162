Operadores para Strings en PHP

PHP permite manipular cadenas de texto (strings) usando operadores específicos para concatenar y comparar cadenas.

Operadores principales para Strings

  • . → Concatenación: une dos cadenas en una sola.
  • .= → Concatenación con asignación: añade una cadena al final de otra existente.

Ejemplos


<?php
  $nombre = "Daniel";
  $apellido = "Ramírez";

  // Concatenación
  $nombreCompleto = $nombre . " " . $apellido;
  echo $nombreCompleto; // Daniel Ramírez

  // Concatenación con asignación
  $saludo = "Hola";
  $saludo .= ", ¿cómo estás?";
  echo $saludo; // Hola, ¿cómo estás?
?>
