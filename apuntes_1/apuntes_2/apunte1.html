Comentarios en PHP

Los comentarios son líneas de texto dentro del código que no se ejecutan. Sirven para explicar lo que hace un programa, documentar funciones o dejar notas para otros programadores (o para ti mismo en el futuro).

Tipos de comentarios

En PHP existen tres formas principales de escribir comentarios:

  • De una sola línea: se escriben con // o con #. Todo lo que esté después de estos símbolos será ignorado hasta el final de la línea.
  • De varias líneas: se escriben entre /* ... */. Son útiles para explicar bloques de código o escribir documentación extensa.

Ejemplo básico


<?php
  // Esto es un comentario de una sola línea
  echo "Hola mundo"; // También se puede poner al final de una instrucción

  # Otra forma de comentario corto
  $x = 10; # Esto asigna 10 a la variable $x

  /* 
     Este es un comentario
     que ocupa varias líneas.
     Se usa mucho para documentar funciones.
  */
  function suma($a, $b) {
      return $a + $b; // Devuelve la suma
  }
?>

¿Por qué usar comentarios?

  • Ayudan a recordar qué hace una parte del código.
  • Facilitan que otras personas entiendan tu trabajo.
  • Son útiles para “desactivar” temporalmente líneas de código durante pruebas.
