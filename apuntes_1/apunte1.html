¿Qué es PHP?

PHP (PHP Hypertext Preprocessor) es un lenguaje de scripts generalista y Open Source, diseñado especialmente para el desarrollo de aplicaciones web. Se integra fácilmente al HTML y se ejecuta en el servidor.

Cómo funciona PHP

El código PHP se incluye dentro de etiquetas <?php ... ?> en archivos HTML. Cuando un usuario solicita la página, el servidor procesa el código PHP y genera HTML que se envía al cliente. Esto significa que el usuario solo ve el resultado final y no el código PHP que lo produjo.


Ejemplo de introducción


<!DOCTYPE html>
<html>
<head>
<title>Ejemplo</title>
</head>
<body>

<?php
  echo "Hola, soy un script PHP!";
?>

</body>
</html>

Ventajas de PHP

  • Extremadamente simple para principiantes.
  • Permite escribir scripts dinámicos mezclando HTML y código.
  • El código se ejecuta en el servidor, garantizando seguridad y control sobre el contenido.
  • Flexible para proyectos simples o avanzados.

Dato: Puedes configurar el servidor para procesar todos los archivos HTML como PHP, haciendo que cualquier página pueda ser dinámica sin que el usuario lo note.
