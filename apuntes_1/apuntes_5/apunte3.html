Proyecto yt_colores

```

yt_colores es una pequeña aplicación CRUD en PHP que permite gestionar una lista de colores (usando las clases de alert de Bootstrap) y sus descripciones. Está pensada como ejemplo práctico para ver operaciones básicas con bases de datos (MySQL), interacción con formularios y una interfaz visual inmediata gracias a Bootstrap 5.

¿Qué hace y por qué es útil?

Permite crear nuevos registros de color, leer la lista completa, editar entradas existentes y eliminarlas. Cada color se muestra en la interfaz usando la clase alert-* de Bootstrap para que el resultado sea visual y directo: si guardas success verás un alert verde, si guardas danger verás rojo, etc. Es ideal para practicar operaciones CRUD y ver inmediatamente el efecto visual de los datos.

Archivos principales y su rol

  • index.php — Panel principal: consulta la tabla y lista todos los colores con controles para editar o borrar; incluye enlace para agregar un nuevo color.
  • agregar.php — Formulario para crear un nuevo color; procesa el POST e inserta el registro en la base de datos.
  • editar.php — Formulario pre-llenado para modificar un color; al enviar hace UPDATE y redirige al listado.
  • eliminar.php — Script que recibe id por GET y borra la fila correspondiente (normalmente tras confirmación).
  • conexion.php (recomendado) — Centraliza la conexión a la base de datos para reutilizar la misma lógica desde todos los archivos.
  • borrame.php — Utilidad o script de pruebas (depuración/ensayo de queries).

Flujo de la aplicación

  1. Abrir index.php → el script consulta tabla_colores y muestra cada fila como un div.alert con íconos para editar o eliminar.
  2. Clic en "Agregar color" → abrir agregar.php, completar formulario y enviar → el servidor inserta el registro y muestra un mensaje (o redirige).
  3. Editar → abrir editar.php?id=... con campos pre-llenados → guardar cambios → el servidor ejecuta UPDATE y vuelve al listado.
  4. Eliminar → eliminar.php?id=... ejecuta DELETE y regresa al listado (ideal con confirmación JS antes de borrar).
```
