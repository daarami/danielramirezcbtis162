Variables en PHP

Las variables en PHP permiten almacenar datos que luego podemos usar y manipular en nuestro código.

Definición y sintaxis

En PHP, todas las variables comienzan con el signo $ seguido del nombre de la variable:

<?php
$nombre = "Daniel";
$edad = 18;
$activo = true;
?>
  • El nombre de la variable debe iniciar con una letra o guion bajo (_).
  • No puede contener espacios ni caracteres especiales.
  • PHP es un lenguaje de tipado dinámico, por lo que no necesitas declarar el tipo de dato.

Tipos de datos comunes

  • String: Texto. Ejemplo: $nombre = "Daniel";
  • Integer: Números enteros. Ejemplo: $edad = 18;
  • Float: Números decimales. Ejemplo: $precio = 45.99;
  • Boolean: Valores true/false. Ejemplo: $activo = true;
  • Array: Colección de valores. Ejemplo: $colores = ["rojo", "verde", "azul"];

Uso de variables

Puedes usar las variables para mostrar información, realizar cálculos o concatenar textos:

<?php
echo "Hola, mi nombre es " . $nombre;
$total = $edad + 5;
echo "En 5 años tendré $total años";
?>
